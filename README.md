Programa en Python que implementa las operaciones CRUD
sobre una boleta de compras usando funciones.

## Funciones
- `agregar_producto(nombre, precio)` — CREATE
- `mostrar_boleta()` — READ
- `calcular_total()` — READ
- `actualizar_precio(nombre, nuevo_precio)` — UPDATE
- `eliminar_producto(nombre)` — DELETE
